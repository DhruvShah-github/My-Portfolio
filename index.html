Projects

Smart Parking System

I designed an efficient parking system for commercial lots using STM32 Nucleo boards with the goal of helping drivers save time and reduce frustration during peak hours. The system guides users through entry and exit while showing real-time parking availability via seven-segment displays. Using UART communication between microcontrollers helps reduce congestion and minimize carbon emissions, contributing to a smoother, more sustainable urban experience.

Student Grading Software

Created a Python-based desktop application to help teachers manage and update student grades more efficiently. My goal was to simplify their workflow and reduce the time spent on administrative tasks. Using a Tkinter interface, the software allows for easy editing of grades and visualizes student progress, making grading not only faster but also more organized.

Brass Chess Pieces

Inspired by the idea of blending engineering precision with craftsmanship, this project involved designing and fabricating brass king and pawn chess pieces. After modeling them in Onshape, I used a manual metal lathe and vertical mill to bring the designs to life. The process demanded careful attention to detail in order to achieve clean, symmetrical results that were both functional and refined.

Wooden Clock

For this project, my goal was to manufacture a fully functional wooden clock while gaining hands-on experience with digital fabrication and precision design. The clock was modeled in VCarve and fabricated using a laser cutter, applying techniques such as cutting, engraving, and scoring. This hands-on experience strengthened my manufacturing skills while resulting in a piece that’s both functional and visually appealing.

Rings

Manufactured two rings using a manual metal lathe, followed by polishing for a clean, elegant finish. This process allowed me to refine my precision and control when working with metal.

Chocolate Mould

Designed a chocolate mould in VCarve and manufactured it using vertical and CNC milling machines. This project helped me improve my skills in both digital design and machining techniques.

Keychain

My first manufacturing project, a custom keychain, was designed in VCarve and manufactured using a CNC milling machine, a vertical band saw, and a drill press. This experience was a great start, allowing me to develop essential skills in machining.
